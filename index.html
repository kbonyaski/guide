01

강의 촬영 개요

필기화면과 강사 얼굴 영상을 통합한 강의 제작 시스템
  • 강의 촬영은 웹캠(강사 얼굴), 무선마이크(음성), 태블릿(필기화면)을 통합하여 진행합니다.
  • 얼굴영상과 음성은 메인PC(노트북 또는 데스크탑)에 연결된 웹캠과 마이크를 통해 녹화합니다.
  • 필기화면은 태블릿을 캡처보드로 메인 PC에 연결하여 OBS프로그램을 통해 영상/음성을 통합 녹화합니다.
  • 최종 영상은 필기화면(메인)과 얼굴영상(보조 썸네일) 구성으로 제작됩니다.

03

촬영 세팅

조명 · 카메라 설정
  • 적절한 밝기로 조정: 너무 밝지 않게, 그늘지지 않게 설정
  • 기본 세팅: 정면에 링라이트 1개 + 웹캠 배치
  • 보조 조명: 필요시 측면에 보조광 추가하여 자연스러운 조명 연출
  • 목표: 얼굴을 균일하게 비추어 선명한 영상 확보
배경 설정
  • 허용 배경: 흰색 또는 흐린 무늬의 벽면만 사용 가능
  • 배경 정리: 사물이 비치지 않도록 깔끔하게 정리
  • 구도 유지: 뒷벽으로 막힌 구도 유지하여 시선 분산 방지
  • 크로마키/배경지: 일정하지 않은 환경일 경우 배경 통일용으로 사용 가능
의상 가이드

🚫 금지 사항

  • 흰색 옷 착용 금지 (배경과 구분 어려움)

✅ 권장 의상

  • 중간톤 또는 짙은색 단색 의상 착용

⚠️ 주의사항

  • 잔잔한 패턴은 가능. 강한 패턴 지양
  • 과도한 반사·소음 유발 액세서리 자제
오디오 — 무선 핀마이크 구성 및 설치

설치 방법

  • C형 송신기를 메인 PC에 연결
  • 마이크는 가슴 중앙부에 부착
  • 케이블 및 옷과의 마찰 소음이 발생하지 않도록 주의
  • 마이크와 입 사이 거리 유지 (약 15-20cm 권장)
  • 수신기와 마이크 페어링 상태 확인
  • 배터리 충전 상태 확인

오디오 체크포인트

  • 수신기 LED 불빛 확인 (정상 연결 시 녹색 점등)
  • 테스트 녹음 5초 진행 후 음성 들리는지 확인
  • 옷 마찰 소리 발생 여부 체크
  • 주변 배경 소음 확인 및 최소화
  • 송신기·수신기 배터리 충전 상태 확인
  • 음성 레벨 적정 여부 확인 (과도한 소리 왜곡 방지)
토익 LC 강의 — 음원 재생 및 활용
📢 스피커로 음원을 틀면, 무선 핀마이크가 그 소리를 다시 수음하게 되어 하울링이 발생합니다. LC 강의 촬영시에는 이어폰을 이용해 주세요.
구분방식이유
A안 (권장) 메인 PC에서 음원 직접 재생 (OBS에 시스템 사운드 포함) 음질 손실 없이 고음질로 녹음 가능. 마이크로 스피커 소리 재녹음되는 현상 방지
B안 (대체, 비권장) 태블릿 또는 외부 기기에서 음원 재생 + 마이크로 수음 마이크 입력과 겹치면 음질 저하 및 반향 가능성 있음. 테스트 필수

OBS 세팅 가이드 (LC 강의 기준)

1
음원 파일(mp3)을 메인PC에서 재생
2
OBS → 오디오 믹서 설정 → 데스크탑 오디오 활성화
3
강사님 마이크 입력(무선 핀마이크)과 시스템 사운드 모두 체크
4
짧은 테스트 녹화 후: ① 음원과 강사 목소리가 모두 명확하게 들리는지 ② 음량 밸런스(음원 60%, 목소리 100%정도) 확인

01 사용 정보 설정

녹화 최적화, 방송은 하지 않음

02 비디오 설정

초당 프레임 30

01 장면 추가
1
'장면' 섹션에서 '+' 버튼 클릭 → 장면 이름 입력 (필기화면)
2
'소스' 섹션에서 '+' 버튼 클릭 → '비디오 캡처 장치' 선택 → Live Gamer MINI (캡처보드) 선택 → 태블릿 필기화면이 표시됩니다
3
다시 '+' 버튼 클릭 → '비디오 캡처 장치' 선택 → 카메라 찾아서 선택
02 화면 구성
  • 필기화면: 메인 전체화면으로 설정
  • 얼굴영상: 화면 우측하단에 작은 썸네일로 배치
  • 썸네일 크기: 전체 화면의 1/6~1/8 비율 권장
  • 레이아웃 저장: 설정 완료 후 '장면' 저장
03 오디오 입력
  • '소스' 섹션에서 '+' 버튼 클릭
  • '오디오 입력 캡처' 무선마이크 선택
  • 레벨 미터로 입력 신호 확인 (녹색~노란색 영역 유지)
  • 녹화시 과도한 소리(빨간색 영역)는 자제
  • 마이크 속성에서 필터 추가 가능 (노이즈 제거)
04 녹화 설정

설정 → 출력 탭

  • 녹화 형식: mp4 권장 (호환성 우수)
  • 인코더: 하드웨어 인코더 권장 (NVENC 또는 AMD)
  • 비트레이트: 5000Kbps 권장

설정 → 비디오 탭

  • 프레임속도(fps): 30 권장
  • 해상도: 1920×1080 권장
  • 50분 분량 MP4 파일로 약 2GB 수준

영상 출력 지원 태블릿 권장 기기

OBS 기반 강의 녹화·송출 환경 구성 시 안정적으로 영상 출력이 가능한 태블릿 모델을 안내합니다.
영상 출력(USB-C → HDMI)을 지원하는 태블릿 기종은 제한적이므로 아래 권장 모델을 참고하시기 바랍니다.

구분제조사모델명주요 특징비고
안드로이드 Samsung Galaxy Tab S7 / S7+ / S8 / S8+ / S9 / S9+ USB-C 단자를 통한 DisplayPort Alternate Mode(영상 출력) 지원 TabS6 Lite, A시리즈 등은 미지원
Lenovo Tab P11 Pro Gen 2 이상 USB-C 3.1 포트 탑재, 영상 출력 지원 보급형 P11 (Gen 1)은 미지원
애플 Apple iPad Air (4세대이상) / iPad Pro (3세대 이상) USB-C 또는 Thunderbolt 3 포트 탑재, HDMI 출력 가능 Lightning 포트 모델은 불가
윈도우 Microsoft Surface Pro 7 ~ 11 시리즈 USB-C or Surface Connect 포트 기반 외부 영상 출력 지원 OBS 직접 설치 및 녹화 가능
